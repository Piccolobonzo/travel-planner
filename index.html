Domande frequenti

FAQ

Il planner funziona senza internet?

Sì. Una volta generato e scaricato, il file HTML funziona completamente offline. Le mappe richiedono connessione solo per caricare i tile la prima volta.

I miei dati sono al sicuro?

TripCraft non salva nessun dato sul server. Il planner generato risiede solo sul tuo dispositivo. Le modifiche vengono salvate nel localStorage del browser.

Posso usarlo per viaggi multi-tappa?

Sì. Il generator supporta tre modalità: singola città, multi-città (più destinazioni condivise) e tour sequenziale (ogni tappa con la propria logistica).

Cosa succede se le informazioni generate non sono accurate?

L'AI genera un punto di partenza ottimo, ma ti consigliamo sempre di verificare orari e prezzi sui siti ufficiali prima di partire. Il planner è facilmente modificabile.

Posso caricare la mia guida di viaggio in PDF?

Sì. Il generator accetta PDF, CSV, TXT e Markdown come fonti aggiuntive. L'AI integra le informazioni del file con i dati generati.

🗺️ How to use the Travel Planner Generator
Schema v1.1 · Generator v10.0 · Planner v7.0
Flusso di lavoro
0
Add A/R transport (optional)
If you have already booked flights or trains, add them here. Travel days automatically become "transfer" days in the calendar. You can add multiple segments for connecting flights.
1
Configure the trip
Enter title, dates, trip type (single city / multi-city / tour) and legs. Choose the sections to include in the planner. For multi-leg tours add one leg per destination with the expected number of days.
2
Choose style and preferences
Concise — generates slots with free text, ideal for flexibility.
Structured — generates activities with times for each slot (recommended).
Detailed — adds descriptions, prices and links for each activity (uses more tokens).
3
Enter the API key and generate
You need an account on console.anthropic.com with credit ($5 covers many tests). The key is only used in this browser session and is never saved. Generation makes 5-6 API calls in sequence.
4
Download JSON and open it in the Planner
After generation, download the JSON and drag it into planner_template_v7.html. The planner works offline — no server, no account, all data in the browser.
Estimated API cost
Concise: ~$0.03–0.08 for a simple trip
Structured: ~$0.08–0.18 (recommended)
Detailed: ~$0.15–0.35
Cost increases with the number of legs and sections enabled.
In the Planner — what you can do
📅 Calendar — click a day to see the slots. ✏️ to edit the text.
✈️ Journeys — confirm bookings with the green button.
🗺️ Map — filter layers by category. Click 🗺️ on any card to center the map.
📝 Personal notes — every card has a 📝 button to add notes. Saved in the browser.
🏨 Accommodation — select "Confirm booking" to mark the chosen hotel.
🌙 Dark / Light — toggle in the top right. Preference saved in the browser.
Notes on AI-generated data
AI-generated data (restaurants, hours, prices, coordinates) has medium/low confidence. Always verify before travelling: opening hours, updated prices, booking requirements. Coordinates are generally accurate but may have small offsets — use as a starting point.
